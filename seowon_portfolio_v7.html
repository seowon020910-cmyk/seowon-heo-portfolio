Emotion Translation Bureau View full FMP →

Core Question

"Can emotion be translated as a system?"

This project steps away from narrative emotion — no catharsis, no story arc. Emotion is observed, analysed, and output as structural data.

Same input pressure → different outputs. This mirrors how human emotion actually works: not a fixed state, but a dynamic response architecture.

Tools: Midjourney · Kling · CapCut  ·  8–10 min experimental film
Frame: ETB (academic) / SYNFEEL (brand / interface)

Visual Language — Textile as Structure

Lace, silk, organza, pearls, stitch lines — structural metaphors, not aesthetics.

The textile horse / unicorn is emotion made physical: if emotion had a body, what structure would it have? The SYSTEM MODE overlay shifts the film from poetic to analytical mid-sequence.

Four Output States

  • State_01

    Torn

    Rupture under excess load — emotional overload as physical fracture.

  • State_02

    Inflated

    Suppressed emotion swells beyond the structure — distortion through containment.

  • State_03

    Thinned

    Depletion — structure becomes translucent, fragile. Quiet erosion.

  • State_04

    Collapsed

    Total structural failure — the state before any restructuring begins.
